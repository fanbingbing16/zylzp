数组（array）是按次序排列的一组值。每个值的位置都有编号（从0开始），整个数组用方括号表示
数组本质上是一个对象。typeof会把数组识别成一个对象
数组的键名是一组按照次序排列的正整数

这个正整数其实是字符串，因为用[]输入数字的时候会被自动转为字符串，.结构是行不通的，因为点结构不会把数子转为数组


length返回数组的成员个数

最大为2^32-1个

如果把length属性赋值给一个小于当前成员个数的值，数组会自动把多出的成员删除，如果大于当前成员个数的值，数组会用
undefine代替多出的部分，如果length属性被赋值给一个不合法的值会报错

var a = [];

a['p'] = 'abc';
a.length // 0

a[2.1] = 'abc';
a.length // 0

可以为数组添加属性，如果添加的精明不是正整数，length不会有变化，如果是数字，但是却不是正整数或者大于2^32-1的length依旧没有变化（这个情况得是空数组），但是数组里面成功添加了值
